## Quick Start
```bash
# Clone repository
git clone https://github.com/yourusername/trading-bot.git

# Install dependencies
pip install -r requirements.txt

# Set up configuration
python setup.py

# Run the application
python -m src.app
```

## System Requirements
- Python 3.9+
- PostgreSQL 13+
- 4GB RAM minimum
- Stable internet connection
